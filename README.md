# Ubuntu Cleaner

## Installing
Copy and execute following command to install Ubuntu Cleaner
```
wget -qO uc https://raw.githubusercontent.com/enishant/ubuntu-cleaner/1.0/ubuntu-cleaner.sh && sudo sh uc
```

## Usage
To get started with Ubuntu Cleaner, run
```
sudo uc
```
